About

I spent 7 years at Micron Technology building systems for semiconductor operations across Singapore, Boise, Hiroshima, and Taichung. The work covered search, risk, vision, simulation, and shared data workflows. Across programs I led, co-led, or supported, it contributed to $600M+ in aggregate program outcomes.

At AI Singapore, I build ML systems across vision, sequence modelling, RAG, and MLOps. My current work includes an AMD 100E industry project. Outside work, I built Job Hunter SG, a personal product for Singapore job search with RAG matching and fact-checked resume guidance.

I like practical AI work: define the constraint, build the API and UI, add evals and observability, then keep tightening the system once real users start using it.
