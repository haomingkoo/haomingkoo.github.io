# Cinematic landing assets

The homepage looks for these files:

- `keyframe-01-ai-alchemist.png`
- `keyframe-02-live-products.png`
- `kooexperience-scroll-reel.mp4`

The page works without them by using real product screenshots as a fallback. Once the MP4 exists, the homepage uses it as the cinematic reel.

Use the prompts in [`../../docs/cinematic-landing-prompts.md`](../../docs/cinematic-landing-prompts.md).
