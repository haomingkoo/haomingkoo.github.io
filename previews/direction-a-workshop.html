How I Build 03

Show the messy middle,
not just the result.

Every project starts with a question, not a spec. Here’s how Trader Koo went from “can AI read candlestick charts?” to a daily-updating production system.

01 — Question

Can YOLOv8 detect chart patterns?

Started with 200 hand-labelled candlestick charts. Trained a v8-nano model. First accuracy: 34%. Not great.

02 — Failed approach

Raw pixel classification

Tried classifying raw chart images. The model memorised background colors, not patterns. Had to rethink the input pipeline entirely.

03 — Breakthrough

Standardised rendering + augmentation

Rendered all charts with identical styling, added rotation/scale augmentation. mAP jumped from 34% to 78%. Deployed to daily cron.
